# Credit-Card-Financial-Dashboard

Credit Card Financial Dashboard

📌 Overview

This project focuses on analyzing credit card transactions and customer data to generate business insights and track key performance indicators (KPIs). The solution integrates PostgreSQL and Power BI to build interactive dashboards that support data-driven decision-making in the financial domain.

🛠️ Tech Stack

Microsoft Excel – Initial data handling
PostgreSQL – Database management & querying
Power BI – Data visualization & dashboarding
DAX – Calculated measures & KPIs

📊 Data Source


The project uses two structured tables stored in PostgreSQL:

🔹 cc_detail (Transaction Data)

Includes credit card transaction details such as:

Card category, annual fees, credit limit
Total transaction amount & count
Interest earned, utilization ratio
Weekly & quarterly performance metrics
Expenditure type and delinquency status

🔹 cust_detail (Customer Data)

Contains customer demographics and profile data:

Age, gender, income, job
Education & marital status
Loan, car & house ownership
Customer satisfaction score

🔄 Project Workflow

Loaded and structured data in PostgreSQL
Performed SQL queries for data extraction and transformation
Imported data into Power BI using database credentials
Built data model and relationships between tables
Created DAX measures for KPI tracking
Designed interactive dashboards for analysis
